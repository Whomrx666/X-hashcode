# X-hashcode 
![X-hashcode preview](X-hashcode.jpg)

## introduction
X-hashcode is a tool used to ENCODE or DECODE a text into hash code form

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ apt update -y && apt upgrade -y
$ pkg install git
$ pkg install python -y
$ git clone https://github.com/Whomrx666/X-hashcode.git
$ cd X-hashcode
$ python X-hashcode.py
```

## Instructions
- **first**: Install tools according to the instructions above
- **second**: Select one of the hashes you want to use
- **third**: Enter text according to your choice
- **last**: Automatic tools will do the hash according to your choice
